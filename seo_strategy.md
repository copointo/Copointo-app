# SEO Strategy

## In scope
- Public customer-facing web surface for Copointo (`artifacts/copointo`)
- Crawlability, metadata, sitemap, robots, AI-crawler readiness, and social sharing for `copointo.com`
- Admin-domain and `/admin` indexation controls where they can leak into public search results

## Out of scope
- Native mobile app store SEO
- Authenticated in-app user flows beyond crawlability/indexation implications
- Internal admin dashboard pages as ranking targets
- API endpoints that are not intended to be indexed as pages

## Target audience
- Arabic-speaking coffee customers in Oman looking for cafes, ordering, booking, reels, and loyalty features
- Secondary audience: cafe managers reaching dashboard entry points
